Outline of the work

In this report, we use a stochastic, meta-population SEIRCDV model to simulate outbreaks of Lumpy skin disease virus. We contrast different response scenario:

  1. baseline: no response
  2. cull_vacc scenario: mass culling of infected farms, ring vaccination
  3. quarant_vacc: quarantine (local and ring) and vaccination (local and ring)
  4. selcull_quarant_vacc: selective culling (only individuals in \(I\)), quarantine (local and ring) and vaccination (local and ring)
  5. selcull_quarant_vacc_insect scenario: all but mass culling (selective culling, vaccination, quarantine, insecticide)
  6. all: all measures combined

Population setting

Our benchmark suggests simulations for 365 days become prohibitively slow for grid sizes exceeding 40 (i.e., 1600 farms). For the final simulation sets, we aim for 1000 replicates in each setting, so we will use a grid size of 40. We assume a diffusion across neighbouring farms of 1%, i.e. 1% of transmission moves to a different farm nearby.

n_rep <- params$n_rep
duration <- 365
grid_size <- 40
n_pop <- grid_size^2
xy <- make_grid(grid_size)
delta <- make_delta(xy, diffusion = 0.05)

The size of the farms, in number of herds, is drawn using an average herd size of 50, exponentially distributed with a minimum at 20 individuals:

r_herd_size <- function(n) {
  20 + round(rexp(nrow(xy), 1/30))
}

herd_size <- r_herd_size(1e4)

Simulations

General settings

We set the number of replicates, which will apply to each simulation scenario. We also make a compiled version of lsdsim and set up initial conditions.

fast_lsdsim <- compiler::cmpfun(lsdsim)

This function is used to generate one new case in a randomly chosen population, to seed outbreaks:

r_ini_case <- function(n_pop) {
  out <- rep(0, n_pop)
  out[sample.int(n_pop, 1)] <- 1
  out
}

baseline scenario: no response

In these simulations, we use no response.

output_dir <- here::here("outputs", "simulations", "baseline", Sys.Date())
if (!dir.exists(output_dir)) {
  dir.create(output_dir, recursive = TRUE)
}

res_baseline <- vector(mode = "list", length = n_rep)
for (i in seq_len(n_rep)) {
  ## run simulation
  current_sim <- fast_lsdsim(
    grid_size = grid_size, 
    time = duration,
    ini_S = unlist(inputs$ini_S[[i]]), 
    ini_I = unlist(inputs$ini_I[[i]]),
    delta = delta,
    beta = inputs$beta[i],
    sigma = inputs$sigma[i],
    gamma = inputs$gamma[i],
    cfr = inputs$cfr[i],
    quarantine = FALSE,
    vaccination = FALSE, 
    mass_culling = FALSE, 
    select_culling = FALSE,
    insecticide = FALSE
  )
  
  ## save full simulation to file
  out_file <- file.path(
    output_dir, 
    paste0(inputs$label[i], "_baseline", ".qs")
  )
  qsave(current_sim, out_file)
  
  ## store summary
  res_baseline[[i]] <- summary(current_sim)[duration, ]
}
