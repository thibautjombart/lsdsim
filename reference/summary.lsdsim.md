# Summary for lsdsim objects

This function sums compartments across all populations in a
[lsdsim](https://thibautjombart.github.io/lsdsim/reference/lsdsim.md)
object, for each time step. It also reports the number of patches in
'naive', 'response' and 'ring' state at every time step.

## Usage

``` r
# S3 method for class 'lsdsim'
summary(object)
```

## Arguments

- object:

  an
  [lsdsim](https://thibautjombart.github.io/lsdsim/reference/lsdsim.md)
  object
